☕ Urban Brew – Coffee Shop UI Design (Figma)

📌 Project Overview
Urban Brew is a modern coffee shop website UI designed using Figma. The design focuses on creating a cozy, premium café experience through a dark aesthetic, clean layout, and intuitive navigation. It includes key sections like a landing page, menu display, and table booking interface.

🎯 Objective
To design a visually appealing coffee shop website UI
To ensure smooth and intuitive user navigation
To create a minimal yet engaging booking experience
To apply UI/UX principles using Figma

👥 Target Users
Coffee lovers and café visitors ☕
Young adults and students
Professionals looking for a relaxing workspace
Users who prefer online table reservations
Customers who want quick access to menu and ordering

🧩 Website Structure
1. 🏠 Home (Landing Page)
Hero section with coffee imagery
Tagline: “Fresh brews. Cozy Vibes.”
CTA buttons: Order Now, Book a Table
Navigation bar:
Menu
About
Contact
Book Table
2. 📋 Menu Page
Grid/card layout for menu items
Each item includes:
Image
Name
Price
Add to Cart button
Clean and visually balanced structure
3. 📅 Book a Table Page
Reservation form with:
Date & Time selection
Name
Email
Phone number
“Confirm Reservation” button
Side image for better visual hierarchy
4. 📆 Calendar Section
Monthly calendar interface
Helps users visualize availability
Navigation controls (previous/next, today)

🎨 UI/UX Design Highlights
Dark theme with warm coffee tones (brown & beige)
Elegant and readable typography
Consistent spacing and alignment
Clear visual hierarchy
Button-focused interaction design
Card-based layout for menu items

🛠️ Tools Used
Figma – UI/UX design, prototyping, and layout creation
Figma Components – for reusable UI elements
Auto Layout – for responsive alignment
Grids & Frames – for structured design

🔄 Design Process
Wireframing basic layout
Creating high-fidelity UI screens
Applying color palette and typography
Designing components (buttons, cards, forms)
Building consistent layout using Auto Layout

🚀 Future Improvements
Convert design into a functional website
Add interactive prototype with transitions
Include user feedback/testing
Expand menu and ordering system
Add mobile-first design variations

💡 Conclusion
This project showcases a complete UI/UX design of a coffee shop website created in Figma. It highlights strong design fundamentals such as layout structuring, visual hierarchy, and user-centered design.








🏋️ Fitness App UI
This project presents a modern and user-centric Fitness App UI designed using Figma. The primary goal of this design was to create an intuitive, visually appealing, and motivating interface that enhances the overall user experience for individuals pursuing their fitness journey.

The UI focuses on clean layout structuring, effective use of spacing, and a consistent visual hierarchy to ensure that users can easily navigate through different sections of the app. Special attention was given to typography, color contrast, and component alignment to maintain clarity and readability across all screens.

The design includes key screens such as the home dashboard, workout tracking interface, and progress overview. Each screen is crafted to deliver essential information in a minimal yet engaging manner, reducing cognitive load while keeping users motivated and focused.

A strong emphasis was placed on usability and accessibility. Interactive elements such as buttons and navigation components are designed to be easily identifiable and responsive. The layout supports a smooth user flow, allowing users to move seamlessly between features like selecting workouts, tracking activities, and viewing progress.

Additionally, the UI incorporates modern design trends such as soft color palettes, structured card layouts, and balanced whitespace to create a professional and aesthetically pleasing look. The design aims to not only serve functional purposes but also inspire users to stay consistent with their fitness goals.

This project reflects my understanding of UI/UX principles, including user-centered design, visual consistency, and effective information architecture.
