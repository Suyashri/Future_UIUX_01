☕ Urban Brew – Coffee Shop UI Design (Figma)

📌 Project Overview
Urban Brew is a modern coffee shop website UI designed using Figma. The design focuses on creating a cozy, premium café experience through a dark aesthetic, clean layout, and intuitive navigation. It includes key sections like a landing page, menu display, and table booking interface.

🎯 Objective
To design a visually appealing coffee shop website UI
To ensure smooth and intuitive user navigation
To create a minimal yet engaging booking experience
To apply UI/UX principles using Figma

👥 Target Users
Coffee lovers and café visitors ☕
Young adults and students
Professionals looking for a relaxing workspace
Users who prefer online table reservations
Customers who want quick access to menu and ordering

🧩 Website Structure
1. 🏠 Home (Landing Page)
Hero section with coffee imagery
Tagline: “Fresh brews. Cozy Vibes.”
CTA buttons: Order Now, Book a Table
Navigation bar:
Menu
About
Contact
Book Table
2. 📋 Menu Page
Grid/card layout for menu items
Each item includes:
Image
Name
Price
Add to Cart button
Clean and visually balanced structure
3. 📅 Book a Table Page
Reservation form with:
Date & Time selection
Name
Email
Phone number
“Confirm Reservation” button
Side image for better visual hierarchy
4. 📆 Calendar Section
Monthly calendar interface
Helps users visualize availability
Navigation controls (previous/next, today)

🎨 UI/UX Design Highlights
Dark theme with warm coffee tones (brown & beige)
Elegant and readable typography
Consistent spacing and alignment
Clear visual hierarchy
Button-focused interaction design
Card-based layout for menu items

🛠️ Tools Used
Figma – UI/UX design, prototyping, and layout creation
Figma Components – for reusable UI elements
Auto Layout – for responsive alignment
Grids & Frames – for structured design

🔄 Design Process
Wireframing basic layout
Creating high-fidelity UI screens
Applying color palette and typography
Designing components (buttons, cards, forms)
Building consistent layout using Auto Layout

🚀 Future Improvements
Convert design into a functional website
Add interactive prototype with transitions
Include user feedback/testing
Expand menu and ordering system
Add mobile-first design variations

💡 Conclusion
This project showcases a complete UI/UX design of a coffee shop website created in Figma. It highlights strong design fundamentals such as layout structuring, visual hierarchy, and user-centered design.
